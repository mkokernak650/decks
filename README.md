# ConsumerEXP Master Slide Decks

This repository stores the master Markdown source files for ConsumerEXP sales and distribution slide decks. These files are intended to be the source of truth for Codex or another deck-generation workflow to convert into PowerPoint, PDF, or web presentations.

## Deck Guidelines

- Keep each deck to 15 slides or fewer.
- Keep sales slides brief and high-impact.
- Include only a few slides on KPIs and descriptions so the audience is not bored.
- Every deck should end with a Thank You slide using the ConsumerEXP logo.
- Do not include Gemporia or Estate Jewelry Shopping unless later approved.
